Deploy DeepLX on Cloudflare.

## Usage

Development
```
npm i && npm run dev
```

Deploy
```
npm run deploy
```

Alternatively to deploy, open the [Cloudflare dashboard](https://dash.cloudflare.com/), select `Workers & Pages` -> `Create Application` -> `Create Worker` -> `Deploy` -> `Edit Code`, and paste the following code, then click `Save and Deploy`.
